# algo-cram-week

牛客网「[算法笔面试篇 · 笔试模板必刷](https://www.nowcoder.com/exam/oj?topicId=389)」题库（共 147 题）的一周刷题计划 + 可交互动画讲解网页。

面向有 C++ 基础但算法能力薄弱、时间有限、想在一周内对着答案快速建立"题型地图"的人。

## 怎么用

直接用浏览器打开 [`index.html`](./index.html)（双击即可，纯静态页面，不需要装任何东西、不需要联网）。里面是 7 天的入口，每天对应牛客题库里的一段真实题号范围。

### 三轮刷题法

1. **第一轮（配合这个网站）**：看动画演示看懂套路 → 做「填空练习」验证是否真的理解 → 点开「配套真题」里的真实链接，直接看答案/题解，照着思路自己敲一遍代码，调试到 AC → 记一句卡点备注。目标是"看懂 + 跑通"，建立题型地图，不追求独立解出。
2. **第二轮（自己安排）**：不看动画和答案，独立重做一遍第一轮做过的题，卡住了再回来翻对应页面复习。
3. **第三轮（自己安排）**：完全从头独立完成，检验是否真正掌握。

## 每天讲什么

| Day | 分类 | 题号范围 | 核心套路 |
|---|---|---|---|
| 1 | 基础算法（上） | BISHI1–21 | 前缀和、双指针、排序、位运算、模拟 |
| 2 | 基础算法（下） | BISHI22–42 | 桶思想/计数排序、构造、异或性质、巴什博弈、整除分块、随机化（选学） |
| 3 | 贪心算法 + 数学算法（上） | BISHI43–63 | 区间贪心、反悔堆、质数/GCD、递推/斐波那契 |
| 4 | 数学算法（下） + 搜索算法（上） | BISHI64–84 | 快速幂、组合数学、高级数论（选学）、DFS、BFS |
| 5 | 搜索算法（下） + 树图算法 | BISHI85–105 | 二分查找、记忆化搜索、字符串匹配（选学）、树的遍历、并查集 |
| 6 | 树图算法（尾） + 数据结构 | BISHI106–126 | Dijkstra、差分数组、滑动窗口、单调栈、倍增/LCA（选学） |
| 7 | 数据结构（尾） + 动态规划 | BISHI127–147 | 线段树（选学）、最大子段和、01 背包、完全背包、进阶 DP（选学） |

标"选学"的是竞赛/进阶向内容，校招笔试很少直接考，时间紧可以先跳过。

## 页面里有什么

每个套路一张卡片，包含：

- 一句话讲清楚这个套路是什么、什么时候用
- 一段可交互 Canvas 动画（播放 / 暂停 / 单步 / 调速），把算法执行过程可视化
- 一段核心 C++ 模板代码的填空练习，「检查答案」会标红/标绿，支持忽略空格差异和常见等价写法
- 「配套真题」：题号、题目名、难度、直达链接，均来自牛客题库页面实际内容（不是编出来的）

## 目录结构

```
index.html          总览页，7 天入口
day1.html ~ day7.html   每天的动画讲解页（自包含单文件，内联全部 CSS/JS）
_data/problems.json     从牛客题库页面抓取的 147 道题原始数据（题号/标题/难度/链接/分类）
_build/                 生成 day2~day7 页面用的构建脚本（day1.html 是手写模板，其余页面复用同一套引擎）
```

`_data` 和 `_build` 是内部工具/数据，不影响直接打开网页使用；只有想批量改版/加新的一天时才需要。

## 说明

- 所有动画、场景渲染、填空校验都是纯前端 JS，不依赖任何外部 CDN，离线可用。
- 每个页面的打卡进度保存在浏览器本地 `localStorage`，仅保存在你自己电脑上；由于是直接以文件方式打开，各页面的进度相互独立，不会自动汇总到总览页。
- 少数套路（前缀和、双指针）严格来说属于牛客题库的「数据结构」分类，因为足够基础被提前放进了 Day 1，页面里有相应说明。
